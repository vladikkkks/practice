# Система онлайн-бронювання квитків на Рівненську дитячу залізницю

Даний проєкт є клієнтським вебзастосунком (SPA), розробленим у межах практичної роботи з дисципліни «Web-програмування». Застосунок дозволяє переглядати розклад поїздів, обирати місця на інтерактивній схемі вагона та керувати купленими квитками в особистому кабінеті.

**Посилання на сайт:** [https://vladikkkks.github.io/practice/](https://vladikkkks.github.io/practice/)

## Мета проєкту
Закріплення навичок розробки інтерактивних інтерфейсів за допомогою чистих технологій HTML5, CSS3 та Vanilla JavaScript. Основна увага приділена маніпуляціям з DOM-деревом, роботі з LocalStorage та реалізації логіки Single Page Application без використання сторонніх фреймворків.

## Функціональні можливості
* **SPA-навігація:** Перемикання між розділами (Головна, Табло, Вибір місць, Кабінет) реалізовано через керування класами активності секцій в одному HTML-файлі.
* **Інтерактивне табло:** Відображення актуального розкладу руху поїздів для станції Центральна.
* **Валідація дат:** Алгоритм перевірки днів курсування, що обмежує купівлю квитків у будні дні (понеділок-середа) згідно з графіком роботи залізниці.
* **Схема вагона:** Візуалізація компонування місць 1+2 (моделі Pafawag) реалізована за допомогою CSS Grid.
* **Система замовлень:** Вибір вільних місць, автоматичний підрахунок вартості та збереження даних про квитки в LocalStorage.
* **Особистий кабінет:** Можливість перегляду придбаних квитків та функція їх повернення з автоматичним вивільненням місць на схемі вагона.

## Технічний стек
* **HTML5:** Семантична верстка, використання контейнерів для екранів SPA.
* **CSS3:** Flexbox та Grid Layout для адаптивності, CSS-змінні для підтримки корпоративного стилю Укрзалізниці, Keyframes для анімації інтерфейсу.
* **JavaScript (ES6+):** Обробка подій, робота з об'єктом Date, JSON-серіалізація даних для локального сховища.

## Структура репозиторію
* `index.html` — основний файл застосунку, що містить усі в'ю та модальні вікна.
* `style.css` — стилізація інтерфейсу та опис сітки вагона.
* `main.js` — бізнес-логіка: навігація, обробка бронювань та робота з даними користувача.

## Як запустити
Для запуску достатньо відкрити файл `index.html` у будь-якому сучасному браузері. Встановлення додаткових залежностей не потрібне.
